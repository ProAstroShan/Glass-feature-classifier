# Glass-Prediction
I try to analyse a data set of values of the features of glass, to predict the best 2 features suitable for KNN algorithms

I created the correlation ratio algorithm here, while AI created the LOOCV confirmation

Disclaimer: I just reealised i did not normalise the data, so we could be very wrong
here. However, after all this is just a simulation and acts a proof of concept. Just
note that you SHOULD implement a normalisation algorithm and replace training data as that, before you want to run this. Maybe my initial prediction was correct and normalisation skewed the LOOCV results, who knows lol

Conclusion:
Correlation ratio ironically may not correlate and provide the true 2 best features
Though it could be useful in weeding out the worst of the worst features
